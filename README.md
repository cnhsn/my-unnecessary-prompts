# 📂 my-unnecessary-prompts

Welcome! This is a collection of prompts that absolutely nobody asked for, yet here they are. 

They won't change the world, they won't revolutionize your workflow, and they definitely won't make you a millionaire. But hey, they might make you smile (or sigh).

## 🤷‍♂️ Why does this exist?
Because sometimes, efficient prompt engineering is overrated. This repository is dedicated to the art of over-engineering simple questions and asking AI to do things that you could probably do yourself in two seconds.

## 🚀 The "Unnecessary" Collection

You can find each of them in `/prompts/` folder.

## ⚙️ How to Use
1. Find a prompt you don't really need.
2. Copy it.
3. Paste it into an LLM.
4. Regret the API credits you just wasted.

## 🤝 Contributing
Do you have a prompt that is beautifully useless? Perfect! Feel free to open a Pull Request. Just ensure it solves absolutely no major global crises.
